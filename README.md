
## 推荐编程工具

### Claude Code

Claude Code 支持 Claude 订阅和 API 配置两种模式。

- [CC-Switch](https://github.com/farion1231/cc-switch)

不同模型擅长做不同的任务，使用 CC-Switch 可以方便切换 Claude Code 配置 BaseUrl、APIKey.

## 其他工具

### 语音工具
语音输入的速度比手动打键盘快 4 倍，可以使用语音工具来提高效率。

- Typeless

语音输入比较优秀的一个选择，不仅能够把你的语音输入进行转录，而且还可以对内容进行结构化润色。

- 闪电说

有本地语音模型的版本，可以自定义配置语言模型的 API。
